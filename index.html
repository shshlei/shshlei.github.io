Create personal blog with hexo

##1. Prerequisites

1.1 Installing Nodejs, npm and hexo

$ sudo apt-get update && sudo apt-get install nodejs-legacy

hexo init blog

cd blog

sudo npm install

hexo server

ls

vim _config.yml

deploy:
type: git
repository: https://github.com/shshlei/shshlei.github.io.git
branch: master

sudo npm install hexo-deployer-git –save

hexo deploy
```

Now, you can visit the website https://shshlei.github.io/ to enjoy you blog.
